\# E-Commerce Automation



\## Description

Automated test for e-commerce site: search, add product to cart, and verify cart contents using Python, Selenium, and PyTest.



\## Tools

\- Python

\- Selenium

\- PyTest

\- ChromeDriver



\## How to Run

1\. Install dependencies: `pip install -r requirements.txt`

2\. Run tests: `pytest tests/test\_ecommerce.py`

3\. View test output in console.



\## Project Structure

\- `pages/` - Page Object Model classes  

\- `tests/` - Test scripts  

\- `requirements.txt` - Dependencies  

\- `README.md` - Project documentation
